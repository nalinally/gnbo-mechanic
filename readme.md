### このリポジトリの説明

- 学校の自主プロジェクトという授業内の成果物の機械部分です
- 制御部分は[gnbo/control](https://github.com/nalinally/gnbo-control.git)リポジトリにまとまっています

### プロジェクト概要

- 3機のモジュール型ロボットが合体・協調して一つのロボットのように振る舞います
- それぞれのロボットは磁石の力を使って、動的に結合/結合解除を行います


### ファイル構造

- `all5.SLDASM` が最終成果物です
- `allOO.SLDASM`と`halfOO.SLDASM`のアセンブリファイルだけ直下に置いています
  - allが全体設計、halfが全体設計一歩手前くらいの状態です

- あとは機能ごとにフォルダが分けられています
